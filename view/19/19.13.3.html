19.13.4、任务运行状态变化历史记录（task_run_status_history）
字段名 中文名 数据类型 是否主键 不能为空 字段注释
Serial_No 流水号 VARCHAR2(20) TRUE TRUE 流水号序列
Task_Group 任务分组 VARCHAR2(100) FALSE TRUE 任务分组
Task_Id 任务ID VARCHAR2(100) FALSE TRUE 任务ID
Task_From_Status 变化前状态 VARCHAR2(10) FALSE FALSE 变化前状态
Task_End_Status 变化后状态 VARCHAR2(10) FALSE TRUE 变化后状态
Task_Start_Change_Description 状态变化原因 VARCHAR2(100) FALSE TRUE 状态变化原因：
1-正常执行
2-任务执行时间过长
3-业务问题需要处理
4-其他
Task_Execute_Server_Ip 所在服务器IP VARCHAR2(30) FALSE TRUE 所在服务器IP
Task_Status_Change_Time 发生时间 VARCHAR2(20) FALSE TRUE yyyy-MM-dd HH:mm:ss
