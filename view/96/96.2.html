96.2、使用smartdoc生成api文档
无入侵式文档生成工具：smart-doc
https://smart-doc-group.github.io/#/zh-cn/?id=smart-doc
smart-doc用法
在项目中新建配置文件：${项目根目录}/src/main/resources/smart-doc.json
	#smart-doc.json文件内容：
	{
	   "outPath": "绝对路径文件夹或者项目下的相对路径，例如：./target/api-doc"
	}
在pom文件中使用smart-doc插件
	......
	
		2.7.6
	
	......
	
		com.ly.smart-doc
		smart-doc-maven-plugin
		${smart-doc.version}
		 
			./src/main/resources/smart-doc.json  
			${project.description}  
			
				
				
			 
		
		
			
				
				compile
				
					
					html
使用maven命令生成文档
	//生成html
	mvn -Dfile.encoding=UTF-8 smart-doc:html
	//生成markdown
	mvn -Dfile.encoding=UTF-8 smart-doc:markdown
	//生成adoc
	mvn -Dfile.encoding=UTF-8 smart-doc:adoc
	//生成postman json数据
	mvn -Dfile.encoding=UTF-8 smart-doc:postman
	// 生成 Open Api 3.0+,Since smart-doc-maven-plugin 1.1.5
	mvn -Dfile.encoding=UTF-8 smart-doc:openapi
	// 生成文档推送到Torna平台
	mvn -Dfile.encoding=UTF-8 smart-doc:torna-rest
	
	// Apache Dubbo RPC文档
	// Generate html
	mvn -Dfile.encoding=UTF-8 smart-doc:rpc-html
	// Generate markdown
	mvn -Dfile.encoding=UTF-8 smart-doc:rpc-markdown
	// Generate adoc
	mvn -Dfile.encoding=UTF-8 smart-doc:rpc-adoc
	
	// 生成dubbo接口文档推送到torna
	mvn -Dfile.encoding=UTF-8 smart-doc:torna-rpc
