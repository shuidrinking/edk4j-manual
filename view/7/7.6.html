7.6、BizLogger接口
1、需要实现的接口
：com.edk4j.core.log.BizLogger
：用于实施交易过程中的交易记录动作
：com.edk4j.edk4jdemos.log.BizLoggerImpl
: ，如果没有这样设置，那么将会调用com.edk4j.core.log.DefaultBizLoggerImpl，这个默认实现类不能很好的满足你项目的需要。
：

      /**

       * 实施日志记录，该方法的调用，必须放在service方法的finally中

       * @param requestNo 请求编号

       * @param context 本次交易上下文

       * @param bizName BusinessLogic的id

       * @param logDefine  日志定义

       * @param throwable 执行逻辑过程中抛出的异常

       * @param useMillis businessLogic执行耗时

       * @throws Exception

       */

      public void log(String requestNo, Context context, String bizName, LogDefine logDefine, Throwable throwable, long useMillis);
