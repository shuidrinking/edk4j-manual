5.6.1、edk4j内置字符串处理工具
类：com.edk4j.core.util.StringUtil
方法清单：
public static boolean isBlank(CharSequence cs) 判定字符串是否为空
public static String leftPadding(String orgString, int formatLength, String leftPaddingChar, String prefix) 左边填充字符串至目标结果
orgString 原字符串
formatLength 格式化的长度，实际序列号长度不足时左边补leftPaddingChar，超过时截掉[后]面多余长度
leftPaddingChar 左补的字母
prefix 前缀,补完后,再在结果里补充前缀,实际返回结果的长度=prefix.length+formatLength
public static String rightPadding(String orgString, int formatLength, String paddingChar, String prefix) 右边填充字符串至目标结果
orgString 原字符串
formatLength 格式化的长度，实际序列号长度不足时右边补paddingChar，超过时截掉[后]面多余长度
paddingChar 右补的字母
prefix 前缀,补完后,再在结果里补充前缀,实际返回结果的长度=prefix.length+formatLength
public static String parseTemplateByContext(Context context, String templateContent) 用context里的字段的真实值，将字符串中的${}占位符，使用真实值替换
context
templateContent 要被处理的字符串
public static String encodeByMD5(String sourceStr) 对字符串进行md5加密，加密结果为32位小写。
【注意】如果你需要16位大写或者16位小写或者32位大写，那么你需要在项目中自行时间工具类处理。
public static String nvl(Object value) 如果是空值，则以空字符串取代
public static String nvl(Object valueString, String newValue) 如果是空值，则以指定字符串取代
public static String convertChinesePunctuationToEnglish(String sourceString) 将字符串里面的中文全角字符转换为相应的英文字符
public static String capitalize(String src) 首字母大写，首字母后的字符小写
public static String toCamelCase(String src) 下划线组合串转驼峰式字符串,首字母是小写
