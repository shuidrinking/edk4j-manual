5.6.17、edk4j内置Spring bean操作工具
类：com.edk4j.core.util.ApplicationContextUtil
在edk的软编码开发模式中，很少会用到这个工具类。方法清单：
public static Object getBeanByName(String beanName) 根据beanName从spring的IOC容器中获取bean
public static <T> T getBeanByClass(Class<T> type) 根据Class，从spring的IOC容器中获取bean
public static Object createBeanBySpringAndNotRegistToIOC(String beanName, Class<?> clazz, BeanProperty[] properties) 根据Class和beanName创建一个bean，并且为bean设置指定的属性。创建的bean不会保存在spring的ioc容器里
