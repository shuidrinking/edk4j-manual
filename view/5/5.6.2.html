5.6.2、edk4j内置日期及时间处理工具
类：com.edk4j.core.util.DateUtil
该类中提供的方法相当丰富，能满足开发中的绝大部分日期需求。
方法清单：
public static String getCurrentTime() 获取当前时间,格式为HH:mm:ss
public static String getCurrentTimeWithoutSign() 获取当前时间,格式为HHmmss
public static String getCurrentDate() 获取当前日期,格式为yyyy-MM-dd
public static String getCurrentDateWithoutSign() 获取当前日期,格式为yyyyMMdd
public static String getCurrentDateTime() 获取当前日期及时间,格式化yyyy-MM-dd HH:mm:ss
public static String getCurrentDateTimeWithoutSign() 获取当前日期及时间,格式化yyyyMMddHHmmss
public static String getCurrentDateTime(String format) 按指定格式获取当前时间
public static int daysBetween(Date startDate, Date endDate) 计算两个日期相差的天数
public static long minutsBetween(Date startTime, Date endTime) 计算两个时间相差的分钟数
public static String getTheFirstDateOfThisMonth(String format) 返回本月的第一天日期，格式如果为空时返回yyyyMMdd格式的
public static String getTheLastDateOfThisMonth(String format) 返回本月的最后一天日期，格式如果为空时,默认为yyyyMMdd格式的
public static int getTheLastDateNumOfMonth(Date date) 返回参数所在月的最后一天号数（1-31）
public static Date getTheLastDateOfMonth(Date date) 返回参数所在月的最后一天日期
public static int getTheLastDateNumAfterMonths(Date date, int monthsAfterThisMonth) 获取距离参照日期date 有monthsAfterThisMonth（可为负数）个月的那月最后一天日期的号数（1-31）
public static Date getTheLastDateAfterMonth(Date date, int monthsAfterThisMonth) 获取距离参照日期date 有monthsAfterThisMonth（可为负数）个月的那月最后一天日期
public static String getDateAfterDays(int days,String format) 获取距离当前日期指定天数（可为负）的日期
public static String getDateAfterMonths(int months, String format) 获取距离当前日期指定月数（可为负数）的日期
public static String getDateAfterYears(int years, String format) 获取距离当前日期指定年数（可为负数）的日期
public static Date parseToDate(String date, String format) 将字符串格式的日期，根据格式，转化为Date对象
public static String formatDateTime(Date date, String format) 根据参数格式显示格式化日期时间
public static String convertTimeFormat(String datetime, String oldFormat, String newFormat) 按新格式，把旧格式的日期重新格式
public static String getDifferentTime(int hour, int minute, int second, String timeFormat) 取得指定时间间隔后的系统时间
示例：
getDifferentTime( 1, 2, 3,"yyyyMMddHHmmss")使用yyyyMMddHHmmss格式输出1小时2分3秒后的系统时间
getDifferentTime( -24, 0, 0,"yyyyMMdd")使用yyyyMMdd格式输出1天前的系统日期
public static String getDifferentDate(int days, String timeFormat) 取得若干天(正数为当前日期之后，负数为当前日期之前)的系统日期
public static String[] getCurrentHHmmssArray() 获取当前时间的时分秒数组，返回的结果内容为{HH,mm,ss}
