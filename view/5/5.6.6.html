5.6.6、edk4j内置文件处理工具
类：com.edk4j.core.util.FileUtil
方法清单：
public static boolean writeTextWithUTF8(String fileName, String content) 以UTF-8编码的方式将文本写入文件。要求fileName为文件的绝对路径
public static boolean writeText(String fileName, String content, String encoding) 以指定编码的方式将文本写入文件 fileName为文件绝对路径
public static String readTextWithUTF8(String fileName) 以utf8编码方式读出文本文件，并以字符串方式返回
public static String readText(String fileName, String encoding) 以指定编码的方式读出文本文件，并以字符串方式返回
public static File[] scanDirectory(String absPath) 扫描指定目录的文件并返回文件列表（不会递归扫描子文件夹）
public static Resource[] resolveConfigLocations(String... locations) 提取指定路径表达式下的所有文件，例如：”classpath∗:∗∗/∗.xml"会加载构建路径下的所有xml文件
public static InputStream getInputStream(String filePathName) 获取文件流
public static byte[] getContent(String filePathName) 获取文件字节数据
