5.6、edk4j内置序列化及流处理工具
1、序列化工具
类：com.edk4j.core.util.SerializeUtil
该类是在序列化和反序列化数据的，处理用于网络传输的数据，方法清单：
public static byte[] serialize(Object object) 将对象序列化
public static Object unserialize(byte[] bytes) 将byte数组反序列化
2、流处理工具
类：com.edk4j.core.util.StreamUtil
方法清单：
public static byte[] streamToByteArray(InputStream dataStream) 将inputStream读取并保存到byte数组中
public static String streamToString(InputStream dataStream, String charSetName) 将字符串的stream转化为指定编码方式的字符串
