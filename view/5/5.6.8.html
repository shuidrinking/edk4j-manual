5.6.8、edk4j内置数据检查工具
该工具内的方法，是使用正则表达式实现的，您可以在项目中借助正则表达式开发跟完善的数据检查工具。
类：com.edk4j.core.util.DataCheckUtil
方法如下：
public static boolean isContainChinese(String str) 判定是否包含中文
public static boolean isAllChinese(String str) 判定是否全部为中文
public static boolean isLetterDigit(String str) 判定是否由26个英文字母（不限大小写）加数字组成
public static boolean isAccount(String str) 判定是否以字母（无论大小写）开头，以字母、数字、下划线组成
public static boolean isAccountx(String str) 验证是否以字母开头包含字母数字和一些特殊符号（点、横杠、下划线）
public static boolean isInteger(String str) 判定是否为整数
public static boolean Is2FixedDecimal(String str) 判定是否为保留两位小数的正小数
public static boolean IsIPV4(String str) 判定是否正确的4位IP地址
