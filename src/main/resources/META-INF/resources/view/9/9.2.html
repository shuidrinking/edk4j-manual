9.2、请求数据的提交格式
提交格式1：form格式提交

	<input type="text" name="fieldId"/>
	<input type="file" name="fieldId"/>


	<input name="listxId.field1Id"/>

	<input name="listxId.field2Id"/>

	......

	<input name="listxId.fieldNId"/>
提交格式2：url中以键值对格式提交
（1）“键”与“值”用=连接；
（2）“键值对”之间用&连接。
https://someUrl?key1=value1&key2=value2&key3=value3...
提交格式3：json格式提交
{
	"someFieldId" : "fieldValue",
	"someListxId" :[
		{"field1":"value1"..."fieldN" : "valueN"},
		...
		{"field1":"value1"..."fieldN" : "valueN"}
	]
}
提交格式4：xml格式提交

<xml>
	<dataCell id="someDcId" value="someValue"/>
	<listx id="someListxId">
		<mapx>
			<dataCell id="f1" value="value1-1"/>
			<dataCell id="f2" value="value1-2"/>
			...
			<listx id="subListx">......</listx>
		</mapx>
	</listx>
</xml>




<xml>
	<someDcId><![CDATA[someValue]]></someDcId>
	<someListxId>
		<mapx>
			<f1><![CDATA[value1-1]]></f1>
			<f2><![CDATA[value1-2]]></f2>
			...
		</mapx>
	</someListxId>
</xml>
