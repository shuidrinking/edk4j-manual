95.4、在配置文件中设置从环境变量获取参数
1、配置手法
	#在application.properties、application.yml、bootstrap.yml等配置文件中均可这样配置
	#语法范式：some-item=${ENV_KEY:default-value}
	#例如：
	server.port = ${SERVER_PORT:9999}
	spring.datasource.druid.url=${DATABASE_URL : jdbc:mysql://192.168.0.188:3306/edk4j_star_product?serverTimezone=GMT%2B8&useSSL=true&useUnicode=true&characterEncoding=UTF-8}
2、在启动命令中设置配置值
	#在启动命令中使用-D设置环境变量
	#语法范式：-Dsome-key=some-value
	#例如：
	java -jar ..... -DSERVER_PORT=9999
3、在k8s中设置环境变量
	#按照k8s的规范在相应的页面中设置即可
