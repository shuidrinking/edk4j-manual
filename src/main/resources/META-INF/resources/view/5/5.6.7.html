5.6.7、edk4j内置数学计算工具
金额精确计算，全部要用BigDecimal进行处理，除此之外，本类中还提供了其他数据写计算工具方法。
类：com.edk4j.core.util.MathUtil
方法清单：
public static String subtract(String subtrahendValue, String minuendValue) 用BigDecimal对数值相减
subtrahendValue 被减数
subtrahendValue 减数
public static String add(String value1, String value2) 提供精确的加法运算。
value1 被加数
value2 加数
public static String multiply(String value1, String value2) 提供精确的乘法运算
value1 被乘数
value2 乘数
public static String div(String value1, String value2, int scale) 提供（相对）精确的除法运算。当发生除不尽的情况时，由scale参数指定精度，以后的数字四舍五入。
value1 被除数
value2 除数
scale 表示表示需要精确到小数点以后几位
public static String round(String value, int scale) 提供精确的小数位四舍五入处理
value 需要四舍五入的数字
scale 小数点后保留几位
public static int convertsToInt(String value) 提供精确的类型转换(Int)不进行四舍五入
value 需要被转换的数字
public static String returnMax(String value1, String value2) 返回两个数中大的一个值
value1 需要被对比的第一个数
value2 需要被对比的第二个数
public static String returnMin(String value1, String value2) 返回两个数中小的一个值
value1 需要被对比的第一个数
value2 需要被对比的第二个数
public static int compareTo(String value1, String value2) 精确对比两个数字，如果两个数一样则返回0，如果value1比value2大则返回1，反之返回-1
value1 需要被对比的第一个数
value2 需要被对比的第二个数
public static String formatToInt(String value) 格式化为带千分位的整型
public static String formatFixedTo2(String value) 格式化为带千分位的数字，保留两位小数
public static String formatFixedTo4(String value) 格式化为带千分位的数字，保留4位小数
public static String formatFixedTo2AndAddPercent(String value) 格式化为带千分位的数字，保留2位小数，尾部带百分号
public static String formatFixedTo4AndAddPercent(String value) 格式化为带千分位的数字，保留4位小数，尾部带百分号
public static String percentFormatFixedTo2(String value) 格式化为百分比，保留2位小数，会将原值乘以100后格式化，返回结果的尾部带百分号
public static String percentFormatFixedTo4(String value) 格式化为百分比，保留4位小数，会将原值乘以100后格式化，返回结果的尾部带百分号
