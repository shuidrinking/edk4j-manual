5.6.14、edk4j内置本机网络工具
1、本机网络属性获取
类：com.edk4j.core.util.NetUtil
该工具类，只能获取本机网络属性，方法如下：
public static List<HostNetInfo> getHostNetInfoList() 获取本机mac及对应ip的列表
2、获得本服务器IP、机器名称等关键信息
类：com.edk4j.core.util.ServerInfoUtil
方法如下：
public static ServerInfo getServerInfo() 获得服务器信息
public static void DebugPrintLocalInfo() 打印本机信息
