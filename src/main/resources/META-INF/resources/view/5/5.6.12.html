5.6.12、edk4j内置压缩及解压工具
1、字符串压缩解压工具
类：com.edk4j.core.util.DataCompressUtil
该类是字符串的压缩和解压工具，用于在网络传输大数据的场景中，压缩以便节省传输成本。
方法清单:
public static Object getStringCompressedByte(Object unCompressedObject, String charSet) 按照指定编码，压缩字符串
如果你的项目中没有配置压缩线，则默认压缩线是15K，如果传入的字符串大于15K则会压缩，然后返回压缩后的byte数组，否则将字符串返回！
public static Object getUncompressedString(Object compressedObject, String charSet) 按指定编码解压字符串
如果传入的参数是String，则无需解压，原样返回，如果是byte数组，则解压缩为String字符串
2、文件压缩解压工具
类：com.edk4j.core.util.FileCompressUtil
方法清单：
public static void unCompressZip(String archiveFilePath, String unCompressedToPath) 解压缩zip文件
@param archiveFilePath 压缩包地址
@param unCompressedToPath 解压后保存地址
public static void compressFolder(String sourceFolderPath, String targetFilePath, String archiveFormat) 压缩文件夹
@param sourceFolderPath 要被压缩的目录
@param targetFilePath 压缩后的结果文件绝对路径文件名
@param 压缩文件的格式，目前支持 jar, zip, 7z, tar
public static void compressFiles(List<String> sourceFilePathList, String targetFilePath, String archiveFormat) 压缩多个文件
@param sourceFilePathList 多个原文件的地址列表
@param targetFilePath 压缩后的结果文件绝对路径文件名
@param archiveFormat 压缩文件的格式，目前支持 jar, zip, 7z, tar
public static void compressOneFile(String sourceFilePath, String targetFilePath, String archiveFormat) 压缩单个文件
@param sourceFilePath 要被压缩的文件
@param targetFilePath 压缩后的结果文件绝对路径文件名
@param archiveFormat 压缩文件的格式，目前支持 jar, zip, 7z, tar
