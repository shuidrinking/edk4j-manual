5.6.11、edk4j内置图像处理工具
该工具一般用于生成图形验证码等场景。
类：com.edk4j.core.util.ImageUtil
方法清单：
public static BufferedImage createSimpleImage(String seed, int perCharWidth, int height) 生成简易图片
seed ：图片文字内容
width ：每个字母的宽度
height：图片高度
public static BufferedImage createStandardImage(String seed, int perCharWidth, int height) 根据文字内容生成标准图片
seed ：图片文字内容
width ：每个字母的宽度
height：图片高度
public static BufferedImage createComplexImage(String seed, int perCharWidth, int height) 根据文字内容生成复杂模式图片，文本随机倾斜
seed ：图片文字内容
width ：每个字母的宽度
height：图片高度
public static String getImageEncodeString(BufferedImage, bufferedImage) 获得图片的编码，在浏览器中可以使用下面的方法显示图片
<img src="data:image/png;base64,本方法获得的内容"/>
public static void writeImageToHttpResponse(BufferedImage bufferedImage, HttpServletResponse response) 将图片写到客户端，如果不用上面的方法显示图片，就用这个方法，这是传统方法
public static BufferedImage createThumbImageBuffer(String originalFileAbsPath, int minWidth) 按照指定像素的宽度生成缩略图，获取缩略图BufferedImage
public static boolean createThumbImageAndSave(String originalFileAbsPath, String thumbFileAbsPath, int minWidth) 按照指定宽度生成缩略图并保存到指定位置，可覆盖
