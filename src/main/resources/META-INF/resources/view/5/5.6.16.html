5.6.16、edk4j内置数据库连接工具
用于需要自己在action类中使用数据库连接的场景中。
类：com.edk4j.core.repository.jdbc.DBAccess
方法清单：
public static Connection getConnectionFromDataSource() 从系统定义的数据源中获得数据库连接
这里获得的数据源，其自动提交属性已经被关闭，所以需要你代码里commit
public static void cleanResources(PreparedStatement pstmt) 关闭PreparedStatment
public static void cleanResources(PreparedStatement pstmt, ResultSet resultSet) 关闭PreparedStatement 、ResultSet
public static void releaseConnection(Connection connection) 释放数据库连接，释放后，其自动提交属性被打开
