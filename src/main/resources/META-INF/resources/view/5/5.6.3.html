5.6.3、edk4j内置Json及Xml格式化及反格式化工具
本节所列的工具类，在接口调用、报文处理时可广泛使用。

xmlEncoding 生成结果的字符串编码格式
public static String modelToXml(Object objectString xmlEncoding) 将实体对象转化为xml字符串，结果被添加缩进和回车美化
object 实体对象

public static String modelToXmlWithoutIndent(Object object) 将实体对象转化为xml字符串，结果未做任何美化
object 实体对象
xmlEncoding 生成结果的字符串编码格式
public static String modelToXmlWithoutIndent(Object object, String xmlEncoding) 将实体对象转化为xml字符串，结果未做任何美化

2、Json和实体类转换工具
类：com.edk4j.core.util.JsonParser
方法清单：
public static String modelToJSON(Object obj) 将实体类对象转化为json字符串，字符串有添加缩进和回车美化
public static String modelToJSONWithoutIndent(Object obj) 将实体类对象转化为json字符串，字符串没有添加任何美化
public static <T> T JSONToModel(String jsonStr, Class<T> clazz) 将json字符串转化为clazz定义的实体类对象
public static <T> List<T> JSONToList(String jsonStr, TypeReference<List<T>> type) 将json字符串转换为某类型实体类对象的列表
public static <K, T> Map<K, T> JSONToMap(String jsonStr, TypeReference<Map<K, T>> type) 将json字符串转化为指定的map，例如map<String,String>
public static <K, T> List<Map<K, T>> JSONToListMap(String jsonStr, TypeReference<List<Map<K, T>>> type) 将json字符串转化为指定的map构成的list
3、Json和xml互转工具
类：com.edk4j.core.util.JsonAndXmlParser
方法清单：
public static String doParseXmlStreamToJsonStringUseJackson(InputStream xmlStream, Class<?> clazz) 将xml字符串的流转化为json字符串，这里需要通过类定义中专，也就是说，传入的xml字符串的流，先转化为clazz的对象，再通过这个对象转化为json字符串
public static String doParseXmlStringToJsonStringUseJackson(String xmlString, Class<?> clazz) 将xml字符串转化为json字符串，这里需要通过类定义中专，也就是说，传入的xml字符串，先转化为clazz的对象，再通过这个对象转化为json字符串
public static String doParseJsonStringToXmlStringUseJackson(String jsonString, String encoding, Class<?> clazz) 将json字符串转化为xml字符串
这里需要通过类定义中专，也就是说，传入的json字符串，先转化为clazz的对象，再通过这个对象转化为xml字符串
