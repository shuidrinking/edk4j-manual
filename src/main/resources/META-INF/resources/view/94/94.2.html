94.2、spring-boot2.7开始废弃spring.factories
1、spring-boot2.7弃用，spring-boot3删除
spring-boot2.7中已经申明废除，但是仍然可以使用，从spring-boot3开始彻底删除。
官方说明：

1、Loading auto-configurations from spring.factories is deprecated. See above for more details.
2、The Micrometer team moved the binders to a separate micrometer module named micrometer-binders and deprecated the old binders. To prevent split packages, the imports have changed, too. If you are using the old binders, please adjust your imports from io.micrometer.core.instrument.binder to io.micrometer.binder. Spring Boot 2.7 still supports auto-configuration for the binders in the old module, but this support will be removed in Spring Boot 3.0.
2、替代方法
新建文件：src/main/resources/META-INF/spring/org.springframework.boot.autoconfigure.AutoConfiguration.imports
将原spring.factories文件中的触发自动注册的组件写到这个文件中
每行一个类的full qualified name，无需逗号分隔
