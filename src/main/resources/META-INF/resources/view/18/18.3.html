18.3、动态新增或更新mvc

edk4j支持mvc的可插拔式动态更新，可以动态的添加或者更新mvc，而不需要重启系统。

请求Url：http://url:port/youSystemName/system/audit/updateMvc

需要用ajax访问，请求参数样例：

{"auditPassword":"审计密码加密值","mvcConfigFileName":"someMvc.edk4jMvc","requestUrl":"demo/getCheckCode"}

 

参数说明：

auditPassword 必输

mvcConfigFileName非必输

requestUrl非必输

当 mvcConfigFileName为空而requestUrl不为空时，会只更新这一个mvc；当requestUrl为空而mvcConfigFileName不为空时，会更新配置文件中所有的mvc；当两者都不能空时，会更新这一个mvc。

返回如下结果表示处理成功：

{

         "resultCode":"0",

         "resultMessage":"ok",

         "timeStamp":null,

         "businessDataObject":null

}
